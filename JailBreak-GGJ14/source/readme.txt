
1. Download libGDX http://libgdx.badlogicgames.com/
   For simplicity use the setup UI:

   http://libgdx.badlogicgames.com/nightlies/dist/gdx-setup-ui.jar

2. Download and install full ADT bundle
   (Android Developer Toolkit)
   http://developer.android.com/sdk/installing/bundle.html

3. Check for updates and open Android SDK manager
   Download all suggested packages.

4. Import both GGJ14 projects into ADT as eclipse projects

5. Connect your Android device

6. Hit run as 'Android Application'

7. Enjoy!
